키워드 1페이지를 누가 점유하나 → 진짜 경쟁사 검수 → 미팅용 결과지

실데이터 모드

1영업 대상 + 핵심 키워드

키워드는 "그 병원의 환자가 검색할 말"입니다. 2~3개 넣으면, 각 키워드의 검색 1페이지를 누가(플레이스·블로그·웹) 점유하는지 훑어 경쟁 후보를 모읍니다.

병원명과 지역/과목을 넣고 누르면, 그 지역 환자들이 자주 검색하는 조합을 띄웁니다.
ℹ️
네이버 검색 API가 주는 건 플레이스·블로그·웹문서 노출입니다. 리뷰 수, SNS·카페 채널 보유, 광고(파워링크) 집행은 API로 안 나오므로 — 경쟁사를 추린 뒤 플레이스 상세를 직접 확인해 결과지에 적습니다.

2경쟁 후보 검수 — 진짜 경쟁사 2~3곳 확정

점수 = 여러 키워드에 걸친 노출 빈도 + 순위 가중치. 여러 키워드에서 자주·위에 뜰수록 높습니다. 점수는 추천일 뿐, 진짜 경쟁사는 직접 고르세요.

후보 0곳 선택 0곳
📷 파워링크(검색광고) 캡처 분석 선택
네이버에서 키워드를 검색해 광고 영역을 캡처한 뒤 여기에 끌어다 놓으면(또는 클릭해 선택), 광고 중인 병원을 자동으로 읽어 후보에 “광고 집행” 표식으로 추가합니다. 블로그는 안 해도 광고로 승부하는 경쟁사를 잡아냅니다.
여기로 캡처 이미지를 끌어다 놓거나 클릭해서 선택
읽어낸 광고 병원 — 틀린 글자를 고치거나, 광고가 아닌 항목은 ✕로 빼주세요.

3미팅용 결과지 (자동 초안 + 직접 다듬기)

시장 노출 점유 — 한눈 비교 자동

입력한 키워드 시장에서 누가 더 넓게 노출을 쥐고 있는지. 막대 = 이 판 1위 대비 상대 점유 (괄호 안은 절대 점수).

키워드별 상세 점유 자동 ▶ 펼치기

각 키워드 1페이지에서 누가 어떤 형태로 노출되는지. ■칸이 채워질수록 그 키워드를 장악.

플레이스 블로그 웹/홈피
채널 현황 — 경쟁사가 실제로 무엇을 하고 있나 직접 입력

플레이스 상세에서 보유 채널을 체크하고 숫자를 넣은 뒤, 오른쪽에 브랜드 관점의 해석을 적습니다. "확인"을 누르면 원장님께 보여줄 카드로 정리됩니다. 빈 채널은 결과에 안 나옵니다.
